SA 2024

Clonar Repositório:

git clone 'url-do-repositório'

Commitar alterações

git add . (para mandar todas as mudanças)

ou

git add 'arquivo em especifico que queria enviar'

git commit -m "descrição do commit"

git push

Configurando o usuário na máquina

git config user.name "Nome do usuário"

git config user.email "
